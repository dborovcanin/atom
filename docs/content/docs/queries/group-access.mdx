## GET /groups/:id/access

Returns what resources and capabilities are accessible through membership in this group. Use this to preview what an entity will gain before adding it to a group.

## Query parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `resource_kind` | string | — | Filter by resource kind |
| `action` | string | — | Filter by action/capability name |
| `effect` | `allow` \| `deny` | — | Filter by effect |
| `limit` | int | 20 | Results per page (1-100) |
| `offset` | int | 0 | Pagination offset |

## Use cases

| Query | Endpoint |
|---|---|
| What does joining this group grant? | `GET /groups/g1/access` |
| What device access does this group have? | `GET /groups/g1/access?resource_kind=device` |
| Does this group have any deny policies? | `GET /groups/g1/access?effect=deny` |
